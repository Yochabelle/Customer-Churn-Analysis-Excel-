This project analyzes customer churn using Excel. The workbook contains the raw data, cleaned tables, pivot tables, and a churn analysis dashboard built with pivot charts.

## Project Structure
excel_dashboard/  
    └── Analyzing Customer churn.xlsx  

documentation/  
    └── dashboard_notes.md  

## Key Questions Answered
- Which contract type has the highest churn rate?
- Do customers with higher monthly charges churn more?
- Do customers with fewer subscribed services churn more?
- Does payment method influence churn?
- Are short-tenure (new) customers more likely to churn?
- Which customer segments show the highest risk of churn?

## Insights Summary
- Month-to-month contracts have the highest churn.
- Customers with higher monthly charges churn more frequently.
- Customers with few or no additional services churn more.
- Paper Check payment users churn at a higher rate.
- New customers (low account length) show the highest churn probability.

## Tools Used
- Microsoft Excel (Pivot Tables, Pivot Charts)
